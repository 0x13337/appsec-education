Users API

The Users API provides information about the users in the system. In order to use this API you must log in. Information on this process is found below.

Authentication

An example request might look like: 

GET /user HTTP/1.0
Host: test.com
X-Session: 13066b17bbd780d5384a54830c5594e0ba6fc9e2282a08bec17ec6893b593202
X-Signature: 79202a7b8e8e068d331417867bab9b4cdc7f277acae42779982c02688ecb4d94

[... Additional paramaters for the request go here ...]

Login

Endpoint:POST /login
Requires Authentication: No
Required Parameters:

  • username
  • password
Example Success Response: 
{
  "success": true,
  "message": "You have successfully registered. You can now use your account."
}

Get users

Please note: unless your user has "admin" level access, you will only receive your own information from this endpoint.

Endpoint:GET /users
Requires Authentication: Yes
Required Parameters: None
Example Success Response: 

{
  "users": [
        { ... }
  ]
}
